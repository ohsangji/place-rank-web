자주 묻는 질문

궁금한 점이 있으시면 먼저 확인해보세요.

기존 수작업과 뭐가 다른가요?
수작업은 키워드를 직접 떠올리고 → 네이버에 검색하고 → 순위를 눈으로 확인하고 → 엑셀에 타이핑하는 과정을 반복해야 합니다. 업장 하나에 30분 이상 걸리죠. 이지보드는 이 전체 과정을 클릭 한 번으로 자동 완료합니다. 수천 개 키워드를 몇 분 안에 검증하고, 5위 이내 키워드만 엑셀로 정리해줍니다.
왜 키워드를 여러 개로 분산해야 하나요?
예를 들어 "해운대 맛집"이라는 키워드 하나로 500건의 유입이 한꺼번에 들어오면, 네이버는 이걸 비정상 트래픽(작업)으로 판단합니다. 하지만 500개의 서로 다른 키워드로 각각 1건씩 유입되면? 네이버는 다양한 사람이 자연스럽게 검색한 것으로 인식합니다. 키워드 분산 = 자연스러운 유입 = 순위 점수에 긍정적입니다.
정말 완전 무료인가요?
네, 100% 무료입니다. 기능 제한도 없고, 추출 횟수 제한도 없습니다. 회원가입만 하시면 바로 모든 기능을 사용하실 수 있습니다. 숨겨진 결제나 유료 전환 같은 것 없습니다.
업무 시간이 아닐 때도 사용할 수 있나요?
네, 대기열 기능을 활용하면 됩니다. 퇴근 전에 여러 업장을 대기열에 등록해두면, 컴퓨터가 켜져 있는 동안 순서대로 자동 추출합니다. 다음 날 출근하면 엑셀 파일이 준비되어 있습니다.
내 폰 검색 순위와 결과가 다른데요?
네이버는 내 검색 기록, 위치, 방문 이력에 따라 개인화된 결과를 보여줍니다. 그래서 내 폰에서 1위로 보여도 실제로는 아닐 수 있어요. 이지보드는 서버 기준의 객관적인 표준 순위를 제공합니다. 리워드 유입은 다양한 사용자가 접속하므로 표준 순위가 더 정확한 기준입니다.
어떤 업종이든 사용 가능한가요?
네, 음식점/카페부터 미용실, 병원, 학원, 꽃집, 세탁소, 사진관, 숙박업까지 총 36개 업종을 자동 인식합니다. 전국 119개 지역의 랜드마크 DB도 내장되어 있어 어느 지역이든 정확한 키워드를 추천합니다.
블랙리스트 기능이 뭔가요?
이전에 추출한 키워드 엑셀을 첨부하면, 이미 사용한 키워드를 자동으로 제외하고 새 키워드만 찾아줍니다. 같은 업장을 반복 추출할 때 중복 없이 매번 신선한 키워드만 확보할 수 있어서, 매주 키워드를 교체하는 분들에게 필수 기능입니다.
여러 업장을 한꺼번에 작업할 수 있나요?
네, 대기열 기능을 활용하면 여러 업장의 URL과 설정을 등록한 뒤 일괄 시작할 수 있습니다. 목표 수량과 우선 타겟도 업장별로 개별 설정 가능하여 대행사의 대량 작업에 최적화되어 있습니다.
🔵 신지도 / 🟢 구지도가 뭔가요?
네이버 검색 결과에 플레이스가 표시되는 두 가지 형태입니다. 현재는 🔵 신지도 키워드 위주로 접수하시는 분이 대부분입니다. 다만 일부에서는 🟢 구지도 키워드도 함께 섞으면 더 자연스럽다고 판단하여 활용하시는 경우도 있습니다. 이지보드는 둘 다 자동으로 구분해서 표시해줍니다.
